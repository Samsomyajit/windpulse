## [1.0.0] - 2026-01-15

### Added
- Initial release of WindPulse analysis framework
- Wind power density calculation module
- Diurnal pattern analysis
- Heat map and contour visualizations
- Clustering analysis (PCA, t-SNE, UMAP)
- Maintenance optimization algorithms
- Risk-controlled scheduling approach
- Seasonal analysis framework
- Jupyter notebook with complete analysis pipeline
- Support for IMD data format
- Weibull distribution modeling
- Bootstrap analysis for uncertainty quantification
- Markov-based transition modeling

### Features
- Multi-site analysis capability
- 10m hub height analysis
- Seasonal grouping (DJF, MAM, JJAS, ON)
- Multiple visualization types:
  - Contour heat maps
  - Ridge plots
  - Sankey/Alluvial diagrams
  - Risk-loss frontiers
  - Maintenance strip plots
  - Clustering visualizations

### Analysis Outputs
- 60 publication-ready visualizations
- Statistical summaries and tables
- Maintenance recommendations
- Site clustering results
- Power loss reduction estimates
